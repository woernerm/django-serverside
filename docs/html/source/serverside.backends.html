serverside.backends#

This module defines a convenience function for accessing the right backend class.

serverside.backends.get_backend(using: Optional[str] = None) → Optional[serverside.backends.base.BackendBase]#

Selects and returns a backend class depending on the given database name.

Multiple backends may be supported. Therefore, selecting the right one is best done using this convenience function. It uses Django’s DATABASES setting and the given name (parameter using) to return the right backend class.

Parameters

using – The settings.DATABASES entry that shall be used to select a backend.

Returns

Specialized backend class. None, if no appropriate backend was found.
